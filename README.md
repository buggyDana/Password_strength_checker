# Password Strength Checker

A basic Python script that evaluates how strong a password is. It checks for:
* Minimum length of 8 characters.
* At least one uppercase & one lowercase letter.
* At least one number.
* At least one special character.

## How to use
1. Run the script.
2. Enter your password when prompted.
3. The script will tell you what's missing to make your password stronger.
